Nutritious Foods

Overview

Data Set Description

The data I am using to address this question is the "FNDDS Nutrient Values.xlsx" from the United States Department of Agriculture's Food and Nutrient Database for Dietary Studies: https://www.ars.usda.gov/northeast-area/beltsville-md-bhnrc/beltsville-human-nutrition-research-center/food-surveys-research-group/docs/fndds-download-databases/. The dataset contains micronutrient and macronutrient quantities per 100g of various foods. Included in the dataset are many types of foods, such as whole foods (e.g. carrots), generic foods (e.g. chocolate milk), and branded items (e.g. Kellogg cereal). More information about the variable names and measurements is found in the "2019-2020 FNDDS Documentation" file on the USDA website. Each food item is identified by a unique eight-digit 'Food code', and is also described by a 'Main food description' (e.g. 'Soy milk') a WWEIA Category Description (e.g. 'Milk Substitutes'), a four-digit 'WWEIA Category number', which groups similar foods together using the first two digits of the code.

Data Cleaning

Remove Unwanted Entries

The major data cleaning task is to remove the unwanted items from the list of 5624 original food items. I am only interested in raw, whole (single ingredient) foods, rather processed, prepared, or fortified foods. Since most canned foods contain preservatives, all canned foods are removed from the data set. I will use three steps to remove the unwanted food items from the original data set.

  1. Use the first two digits of the WWEIA Category number to remove the following food categories:
  • Milk desserts and sauces = 13
  • Frozen meals, soups, gravies = 28
  • Egg mixture = 32
  • Egg substitutes = 33
  • Yeast breads, rolls = 51
  • Quick breads = 52
  • Cakes, cookies, pies, pastries, bars = 53
  • Crackers, snack products = 54
  • Pancakes, waffles, French toast, other grain products = 55
  • Grain mixtures, frozen meals, soups = 58
  • Meat substitutes = 59
  • Fruits and juices baby food = 67
  • Vegetables with meat, poultry, fish = 77
  • Mixtures mostly vegetables without meat, poultry, fish = 78
  • Salad dressings = 83
  • 'For use' with a sandwich or vegetable = 89
  • Formulated nutrition beverages, energy drinks, sports drinks = 95

  1. Remove foods with a WWEIA Category description containing certain words (e.g. 'sandwich', 'fried', 'Formula').

  2. Remove the remaining unwanted items based on their Main food description.

Calculate Nutrient %DV

The next step is to normalize the nutrient amounts in each food by calculating the Percent Daily Value (%DV), which is the percentage of the recommended Daily Value for each nutrient in a serving of food. The Daily Values are found from the FDA's website (https://www.fda.gov/food/nutrition-facts-label/daily-value-nutrition-and-supplement-facts-labels). Some nutrients do not have defined Daily Values, so those aren't used in this project.

Remove Unwanted Variables

Lastly, we'll remove the variables that weren't normalized by taking the %DV.
